Task – группа из нескольких Activity, с помощью которых пользователь выполняет определенную операцию. Обычно стартовая позиция для создания. Activity  находятся в некотором Task, внутри которого можно открыть активность из другого приложения, и после вернуться к предыдущей активности, так как она находится в одном стеке одного Taska.
