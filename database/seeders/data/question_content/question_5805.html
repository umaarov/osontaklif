Дженерик-класс, в котором нужно переопределить две дженерик функции : areItemsTheSame и areContentsTheSame чтобы понимать, как необходимо сравнивать объекты. Процесс нахождения самого лучшего по времени алгоритма происходит змейками по алгоритму Майерса. Это ускоряет процесс обновление объектов в RecyclerView. Он помогает не писать нам каждый раз adapter.notifyitemchanged(position)
