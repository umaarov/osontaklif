Интерфейс с аннотацией @Dao, в котором описаны все методы, которые будут применяться  для работы с базой данных. Data class с аннотацией @Entity(tablename=»Name»), который будет  представлять таблицу из БД , и абстрактный класс, который наследуется от RoomDatabace и аннотированный аннотацией @DataBase(entities), который будет содержать абстрактное поле с нашим Dao-интерфейсом и будет объединять все наши Entities.
