Это компонент, который наследует дерево объектов, которые инжектится родительским  компонентом. Компонент-родитель должен содержать методы, которые возвращают Factory для каждого из subcomponent, также они должны быть подключены через аннотацию  @Module( subcomponents = [ RegistrationComponent::class,]). Поэтому, необходимо определить фактор-метод в субкомпоненте 

@Subcomponent.Factory 
interface Factory { 
 fun create(): RegistrationComponent 
}

Создать инстанс субкомпонента можно только из инстанса родительского компонента, вызвав у него соответствующий метод.
