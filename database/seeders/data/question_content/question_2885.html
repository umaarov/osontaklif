В этой модели есть 3 основных слоя :
model — бизнес-логика и дата классы, репозитории, источники данных.

view — Activity/Fragment которые отображают данные из бизнес-логики.  

viewModel — мостик между model и view взаимодействие между которыми происходит реактивно  — view ничего не знает о model, и наоборот.
