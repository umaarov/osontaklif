Вся графика описывается в функции, которая обозначается аннотацией @Compose. Для того, чтобы изменить что-то в лейауте, мы вызываем механизм recomposition — заново вызываем  функцию composiable. Значения между двумя состояниями мы передаем при помощи помещения их в → slotTable, при помощи функции remember{ var i =1 }. Также для облегчения работы с состояниями есть класс MuatbleState — при изменении значения экземпляра класса, который находится внутри mutable state, вызывается автоматически recomposition и перерисовывается UI.
