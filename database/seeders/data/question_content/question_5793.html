Gradle превращает kotlin/java → код собирается в DEX →все xml ресурсы генерируют R класс → формируется APK.

Ресурсы отдельно, манифест отдельно, вся логика - в DEX-файлы. У DEX-файла не может быть больше чем 65 000 методов. К примеру, если мы подключим RXJava - там много классов, но из них юзается немного. Для борьбы с эти используем minifyEnabled true, и это даёт возможность чтобы в АПК файлы ложились только те файлы, которые используются.
